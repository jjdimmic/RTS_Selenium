Running this project:

to the the following code you need to install Python 3 as well as the selenium module

to install the selenium module run the following command in the command prompt:

    pip install selenium


to Execute the code simply open the command prompt and enter 

   python rts_selenium.py

NOTE: i have included Chromedrive for Chrome version 99 in the repo make sure you download the compatible version of chrome driver 
      for the version of chrome you have in order to run successfully
